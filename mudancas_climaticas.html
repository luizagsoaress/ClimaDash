Sobre

Esse site é um projeto de estudo com HTML5, CSS, JavaScript, usando Chart.js para desenhar os gráficos, com integração de APIs de clima como a Open-Meteo e a OpenWeatherMap, cujas chaves são protegidas usando um backend em Node.js com Express. Também foi utilizado para criação dos mapas interativos a biblioteca de código aberto Leaflet, e para indicadores ambientais a API World Bank.
